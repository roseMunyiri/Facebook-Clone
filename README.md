## Facebook Clone – Frontend Practice

This project is a frontend practice application inspired by Facebook. It is built using **Next.js**, **TypeScript**, **Tailwind CSS**, and **CSS** to practice modern frontend development.

The goal of this project is to improve my skills in building responsive user interfaces, working with component-based architecture, and using Tailwind for styling.

The project is currently **in progress** and is not yet deployed. I am continuously adding new components and improving the UI as part of my learning process.

### Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* CSS

### Current Features

* Header navigation component
* Reusable UI components
* Basic page layout

### Project Structure

```
src/
 ├── components/
 ├── pages/
 └── styles/
```
