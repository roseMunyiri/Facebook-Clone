## Facebook Clone 

This project is a frontend practice application inspired by Facebook. It is built using **Next.js**, **TypeScript**, **Tailwind CSS**, and **CSS** to practice modern frontend development.

The goal of this project is to improve my skills in building responsive user interfaces, working with component-based architecture, and using Tailwind for styling.

The project is currently **in progress** and is not yet deployed.

---

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* CSS

## How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/roseMunyiri/Facebook-Clone.git
```

### 2. Navigate into the project folder

```
cd Facebook-Clone
```

### 3. Install dependencies

This project uses **pnpm**.

```
pnpm install
```

### 4. Start the development server

```
pnpm dev
```

### 5. Open in your browser

Visit:

```
http://localhost:3000
```
